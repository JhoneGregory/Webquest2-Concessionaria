### Objetivo
Simular uma concessionária aplicando os conceitos de Programação Orientada a Objetos (POO): encapsulamento, herança e polimorfismo.

### Estrutura do Projeto
- `Veiculo.java` — classe base  
- `Carro.java`, `Moto.java`, `Caminhao.java` — classes filhas  
- `Concessionaria.java` — classe principal com o método `main`  

### Como Executar
1. Abra o VS Code e crie uma pasta chamada **Webquest2-Concessionaria**.  
2. Adicione todos os arquivos `.java` abaixo.  
3. Compile com `javac *.java`.  
4. Execute com `java Concessionaria`.
